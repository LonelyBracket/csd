**Specs:**
- Episodes section: ~60% width
- Articles section: ~40% width
- Or: Episodes full width, Articles in sidebar

---

## Sections Breakdown

### 1. Hero / Latest Episode
- Uses `EpisodeCard` featured variant
- Full width
- Prominent play/listen CTA

### 2. Recent Episodes
- 3-4 most recent (excluding featured)
- `EpisodeCard` default variant
- "View All Episodes" link

### 3. Latest Articles
- 2-3 most recent articles
- `ArticleCard` default variant
- "View All Articles" link

### 4. Browse by Topic
- All topics as `TopicTag` components
- Wrapped flex layout
- Click navigates to `/topics/[slug]`

### 5. Featured Guests
- 4-6 guests with most appearances
- `GuestCard` default variant
- "View All Guests" link

### 6. Footer
- Separate component (see footer.md)

---

## Data Requirements

```typescript
interface HomePageData {
  featuredEpisode: Episode;
  recentEpisodes: Episode[];     // 3-4 items
  latestArticles: Article[];     // 2-3 items
  topics: Topic[];               // All topics
  featuredGuests: Guest[];       // 4-6 items
}
```

---

## Loading State

- Skeleton for featured episode (large)
- Skeleton cards for episodes/articles
- Skeleton pills for topics
- Skeleton circles + text for guests

---

## Empty States

Unlikely for homepage, but if no content:
- "Coming soon - check back for our first episode!"
- Subscribe/newsletter CTA
