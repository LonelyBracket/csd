### Article Content Typography

```css
/* Prose styling */
.article-content {
  @apply prose prose-zinc max-w-prose;

  /* Headings */
  h2: text-2xl font-bold mt-12 mb-4
  h3: text-xl font-semibold mt-8 mb-3

  /* Paragraphs */
  p: text-base leading-relaxed mb-4

  /* Lists */
  ul, ol: ml-6 mb-4 space-y-2

  /* Blockquotes */
  blockquote: border-l-4 border-indigo-500 pl-4 italic text-zinc-600

  /* Code */
  code: bg-zinc-100 px-1.5 py-0.5 rounded text-sm font-mono
  pre: bg-zinc-900 text-zinc-100 p-4 rounded-lg overflow-x-auto

  /* Links */
  a: text-indigo-500 underline hover:text-indigo-600

  /* Images */
  img: rounded-lg my-8
}
```

Use Tailwind Typography plugin (`@tailwindcss/typography`).
